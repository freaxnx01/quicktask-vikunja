# Code of Conduct

We are committed to providing a friendly, safe, and welcoming environment for everyone.

## Our standards

- Be respectful and considerate in communication.
- Be constructive when giving feedback.
- Assume good intent and collaborate in good faith.
- Harassment, discrimination, and personal attacks are not acceptable.

## Enforcement

If you experience or witness unacceptable behavior, please report it privately to the maintainer ([@freaxnx01](https://github.com/freaxnx01)) via [GitHub's private reporting](https://github.com/freaxnx01/quicktask-vikunja/security/advisories/new), as described in [SECURITY.md](SECURITY.md). This is a single-maintainer personal project; reports are reviewed and handled as promptly as possible.

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
