# Glossary

**Vikunja**
: A self-hosted task management service. Quicktask creates tasks in a Vikunja
  instance via its HTTP API.

**Project**
: A container for tasks in Vikunja (a place to group related work). Quicktask
  asks you to pick a project so it knows where to create new tasks.

**Task**
: A single to-do item (title + optional details like description, due date, and
  labels). This is the main object Quicktask creates.

**Share Target**
: An Android feature that lets users “Share” text/URLs from other apps into
  Quicktask, which turns that shared content into a new task.

**API Token**
: A secret credential used to authenticate Quicktask to the Vikunja API on your
  behalf. It should be treated like a password.
