# Release Notes

User-friendly summary of changes in each version.

---

## Version 0.3.0

_Released 2026-05-16_

### New Features
- **Batch task creation from multi-line paste** — paste multiple URLs or lines of text into the title field and choose "Create N separate tasks" to turn each line into its own task. URL lines are automatically resolved to their page title; a line that times out or returns an error falls back to the raw URL so the rest of the batch continues uninterrupted.
- **Windows desktop app** — QuickTask now runs as a native Windows desktop application. The app opens in a compact portrait window and supports the full setup → manual entry → project picker → confirmation flow. Downloads are available as a ZIP on the GitHub Releases page alongside the Android APK.

---

## Version 0.2.0

_Released 2026-04-20_

### New Features
- **Favorite projects** — star any project in the picker to pin it to a dedicated "Favorites" section at the top, so your most-used projects are always one tap away.
- **App version footer** — the current version number is shown at the bottom of every screen for easy support and reference.
